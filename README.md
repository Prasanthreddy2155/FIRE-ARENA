# 🔥 Fire Arena

Production-ready Free Fire Esports Tournament Platform built with modern web technologies.

## 🚀 Features

- User Authentication
- Tournament Registration
- Wallet System
- Admin Dashboard
- Tournament Management
- Match History
- Withdraw Requests
- Responsive Design
- Firebase Backend

## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Firebase
- Firestore

## 📌 Project Highlights

- Real-time tournament platform
- Secure authentication
- Mobile-friendly responsive UI
- Separate Admin & User Dashboard
- Production-ready architecture
